Spread оператор
Його по факту основна задача розширювати ітерабельні елементи (масиви, строки) або об'єкти.

Приклад використання: 
Копіювання або об'єднання масивів та об'єктів.

Приклад:
У функції Math.max(...[1, 2, 3]) або [...array1, ...array2].

Rest оператор
Він збирає залишкові елементи в масив або властивості об'єкта.

Збирає всі аргументи функції або залишкові елементи в деструктуризації.
Приклад: ... у функції function sum(...args) або const [first, ...rest] = array.

Ключові відмінності:
Spread: Розширює елементи в окремі частини.
Приклад: Копіювання масиву: [...array].
Rest: Збирає кілька елементів в одну структуру.
Приклад: Збір аргументів функції: function func(...args).
